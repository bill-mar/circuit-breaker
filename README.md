# 🛑 投机心态熔断器 (Circuit Breaker)

一个帮助加密货币交易者**冷静**的 AI 工具。在你梭哈之前，先听听 AI 怎么骂你。

## 功能

- 💀 **地狱级风控官**：用冷酷的数据和心理分析骂醒你
- 🎭 **5种人格风格**：冷血会计、暴躁交易员、心理医生、破产律师、地狱审判官
- 📊 **生存时间换算**：自动计算投入金额相当于几个月的生活费
- 🎲 **高度随机化**：每次回复都不一样，保持新鲜感

## 如何使用

1. 在侧边栏输入你的 API Key（支持 OpenAI/Gemini/DeepSeek 格式）
2. 输入你想买的币种（如 DOGE, BTC, ETH）
3. 输入投入金额（USD）
4. 点击"启动风控扫描"
5. 准备挨骂

## 配置

默认使用 OpenAI 格式的 API，支持：
- OpenAI: `sk-...`
- DeepSeek: `sk-...`
- Gemini 反代: `sk-...`

## 安装依赖

```bash
pip install -r requirements.txt
```

## 本地运行

```bash
streamlit run app.py
```

## 在线部署

本项目已部署在 Streamlit Community Cloud，可直接使用。
